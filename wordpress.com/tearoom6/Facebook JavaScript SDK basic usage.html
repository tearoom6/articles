Summary of most basic usage of Facebook JavaScript SDK.
  • FB.init( {options} )  : initialization
options appId / channelUrl / status / cookie / xfbml
  • FB.getLoginStatus( callback [,bool] )  : check user's login status
  • FB.login( callback, { scope : xxx} )  : login or ask user to permissions
  • FB.logout( callback )  : logout
  • FB.api( path [,method] [,{ params}] ,callback )  : call Graph API
