Environment: Windows 8 + Chrome Eclipse 4.2 / Google Plugin for Eclipse 4.2 / Google App Engine Java SDK 1.7.5 / Google Web Toolkit SDK 2.5.0 I had a problem on developing a web application for Google App Engine. The problem was... (1) Run As -> Web Application (on Eclipse Project) (2) Double-click an URL on an emerging [Development Mode] window. (3) Open with Chrome. then, the message is displayed to install Google Web Toolkit Developer Plugin. (4) Click it to install. but you cannot accomplish it. the message says "this application isn't supported". I found a solution in the website below. I copy the reference to the solution. https://code.google.com/p/google-web-toolkit/issues/detail?id=7778 No.5 1. Make sure Chrome is pined to the task bar 2. Right click on the chrome icon>Properties>Shortcut 3. Add in target: --enable-easy-off-store-extension-install 4. Open chrome and navigate to extensions ( chrome://chrome/extensions/ ) 5. Drag and drop on it the plugin (I've attached it) I was able to download the plugin from here: https://groups.google.com/group/google-web-toolkit/attach/d1c909bf4c5cbfb4/gwt-dev-plugin.crx?part=4&authuser=0 Google App Engine向けのWebアプリ作成中に、問題が発生しました。内容は以下の通りです。 (1) Run As -> Web Application (Eclipse上のプロジェクトの右クリックコンテキストメニュー) (2) [Development Mode] ウィンドウが出てくるので、URLをダブルクリック。 (3) Chromeで開くと、Google Web Toolkit Developer Pluginをインストールするようご指令が。。 (4) インストールしようとクリックすると、「このアプリケーションはこのパソコンではサポートされていません。インストールは無効になりました。」 下のサイトで解決策を見つけたので、該当部分をコピペしておきます。 https://code.google.com/p/google-web-toolkit/issues/detail?id=7778 No.5 1. Make sure Chrome is pined to the task bar (タスクバーにChromeを入れておく。要はショートカットであればいいです) 2. Right click on the chrome icon>Properties>Shortcut (そのタスクバーのChromeを右クリック、さらに出てきたメニューの中の[Google Chrome]で右クリックし、[プロパティ]を選択、[ショートカット]タブを開く) 3. Add in target: --enable-easy-off-store-extension-install (リンク先の末尾に上の文字列を追加、こんな風に) "C:\Program Files (x86)\Google\Chrome\Application\chrome.exe" --enable-easy-off-store-extension-install 4. Open chrome and navigate to extensions ( chrome://chrome/extensions/ ) (Chromeを起動し、アドレス欄に上の文字列を入力し、Enter) 5. Drag and drop on it the plugin (I've attached it) (下のサイトで取得できるcrxファイル(Chrome Extention?)をドラッグドロップすれば、Chromeにプラグインを追加できるよ！) I was able to download the plugin from here: https://groups.google.com/group/google-web-toolkit/attach/d1c909bf4c5cbfb4/gwt-dev-plugin.crx?part=4&authuser=0 ⇒Chromeのメニューとかから、[名前を付けて保存]できます。 改めてGAEアプリの起動確認をすると、今度はちゃんとアクセスできました。
