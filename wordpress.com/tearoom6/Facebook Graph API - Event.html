Facebook Graph API – Event

  • get event information

path: EVENT_ID

method: GET

  • set “maybe” status to event

path: EVENT_ID/maybe

method: POST

permission: rsvp_event

  • set “attending” status to event

path: EVENT_ID/atttending

method: POST

permission: rsvp_event

  • set “declined” status to event

path: EVENT_ID/declined

method: POST

permission: rsvp_event

  • get one’s event status

path: EVENT_ID/STATUS/USER_ID

method: GET

permission: user_events (in case user don’t open one’s event status)

  • create event

path: PAGE_ID/events

method: POST

params: name* start_time* end_time description location location_id

permission: create_event

  • modify event

path: EVENT_ID

method: POST

params: name start_time end_time description location location_id

permission: create_event

  • delete event

path: EVENT_ID

method: DELETE

permission: create_event
